ONETAILOR TOOLKIT - USABILITY & LEAD FOLLOW-UP IMPROVEMENTS

==================================================

1. RECENTLY USED TOOLS
   ==================================================

Add a smart "Recently Used" section to the Home Dashboard.

Requirements:

* Track every tool opened by a user.
* Display the last 3-4 tools used.
* Most recent tool appears first.
* One tap opens the tool again.
* Persist history per user account.
* If user has no history, hide section.

Example:

Recently Used

1. Measurement Checker
2. Fabric Requirement
3. Price Smartly
4. Card Generator

This should reduce navigation friction and improve tool engagement.

==================================================
2. PRE-UNLOCK LEAD VISIBILITY
=============================

All information entered during the pre-unlock stage must be visible to Admin.

Capture and display:

* Full Name
* Phone Number
* WhatsApp Number
* Email
* Business Name
* Business Type
* City
* Registration Date
* Last Activity Date
* Tools Viewed
* Tools Used
* Conversion Status
* Subscription Status

Do not discard any lead information.

3. LEAD PRIORITY SCORING
========================

Create a Lead Score system.

Automatically rank leads by likelihood to convert.

Scoring examples:

* Registered recently
* Opened multiple tools
* Added client records
* Returned multiple times
* Viewed pricing pages
* Started upgrade process

Lead score examples:

Hot Lead
Warm Lead
Cold Lead

Admin dashboard should sort by highest priority first.

6. FOLLOW-UP AGENT LOGIN
========================

Create new user role:

Permissions:

Can View Leads
Can Update Lead Status
Can Add Notes
Can Send WhatsApp Messages
Can View Assigned Leads

Cannot:

Access Payments
Access Licenses
Access Settings
Access Users
Access Admin Configuration

This is a restricted CRM-only login.

8. WHATSAPP QUICK ACTION
========================

Add WhatsApp button beside every lead.

One click opens WhatsApp chat.

Use configurable templates.

Examples:

Template 1:
Hello {{name}}, thank you for joining OneTailor Toolkit. We noticed you have not completed your upgrade. Can we help?

Template 2:
Hello {{name}}, your tailoring toolkit account is ready. Upgrade today to unlock all professional tools.

Admin should be able to create, edit and delete templates.

==================================================
9. AUTOMATED FOLLOW-UP QUEUE
============================

Create follow-up automation system.

Rules:

If lead has not converted after 24 hours:

Create follow-up task.

If lead has not converted after 48 hours:

Create second follow-up task.

If lead has not converted after 72 hours:

Create final follow-up task.

Tasks should appear in agent dashboard.
